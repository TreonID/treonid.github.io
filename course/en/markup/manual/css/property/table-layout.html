table-layout property

In some cases, the browser may shrink cells even if they are given a width. The table-layout property allows you to tell the browser that table cells should always be of a given width.

Syntax

selector { table-layout: fixed | auto | inherit; }

Values

Value Description
auto Browser can resize cells.
fixed Browser won't resize cells.
inherit Inherits a parent value.

Default value: auto.

padding-bottom property

The padding-bottom property sets a bottom padding of an element. The property value is any size units.

Syntax

selector { padding-bottom: value; }

Example

Let's make the 30px bottom padding:

<div id="elem"> some text... </div> #elem { padding-bottom: 30px; width: 300px; border: 1px solid black; text-align: justify; }

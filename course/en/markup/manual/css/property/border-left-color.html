border-left-color property

The border-left-color property specifies a color of a left border. The value of the property is any color units. Default value: same color as the block text has.

Syntax

selector { border-left-color: color; }

Example

Let's set the left border to red:

<div id="elem"></div> #elem { border-left-color: red; border-left-width: 1px; border-left-style: solid; background: #f9f9f9; width: 300px; height: 100px; }

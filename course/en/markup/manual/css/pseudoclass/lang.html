lang pseudo-class

The lang pseudo-class selects text based on the language specified in the lang attribute. The following values ​​can be used as a language: ru — Russian, en — English, de — German, fr — French, it - Italian and so on.

Syntax

selector:lang(language) { }

Example

Let's put different quotes for different languages ​​(we use quotes):

<p> Quote in English: <q lang="en">То be or not to be</q>. </p> <p> Quote in German: <q lang="de">Versuche die Gotter nicht</q>. </p> q:lang(en) { quotes: "\201C" "\201D"; } q:lang(de) { quotes: "\201E" "\201C"; }
